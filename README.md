## 📱 Features

- 🔍 Smart Search (dynamic filtering)
- 🥗 Veg Mode Toggle (animated)
- 🍱 Food Categories (horizontal scroll)
- ⭐ Recommended Section (ratings, offers, delivery time)
- 📄 Item Details Screen
- 🛒 Add/Remove Cart Controls (Provider)
